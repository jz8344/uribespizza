# Reglas del Proyecto Uribe's Pizza

## Idioma de Commits
- Todos los mensajes de commit deben ser redactados en español, permitiendo unicamente términos estándar de Git/Conventional Commits como prefijos (`feat:`, `fix:`, `deploy:`, `chore:`, `refactor:`), pero con toda la descripción en español (ejemplos: `feat: Primer commit del servidor Vue 3`, `chore: Eliminación de archivo netlify.toml`, `fix: Actualización de vistas y estilos`).
